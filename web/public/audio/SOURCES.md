# Audio Sources

## Backing Tracks (`tracks/`)

Add royalty-free chillhop/lo-fi tracks here. Good sources:

- <https://pixabay.com/music/search/lofi/> — free with attribution
- <https://freemusicarchive.org/> — filter by CC license
- <https://www.bensound.com/> — free for non-commercial with attribution

After adding files:
1. Drop `.mp3` files in `web/public/audio/tracks/`
2. Add each filename to `tracks.json`
3. Document name, artist, license, and URL in this file
